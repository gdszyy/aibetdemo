# Sportradar UOF 文档

本项目实现了 [Sportradar Unified Odds Feed (UOF)](https://docs.sportradar.com/uof) 用于体育数据。处理体育数据逻辑（WS handler、市场/赔率/specifier 处理、比赛状态、时钟、比分等）时，**始终对照 Sportradar UOF 文档交叉参考**：

| 主题 | 参考 |
|------|------|
| 消息（OddsChange 等） | [UOF Messages](https://docs.sportradar.com/uof/data-and-features/messages/event) |
| 市场映射与 specifiers | [Market Mapping](https://docs.sportradar.tv/display/BD/UOF%2B-%2BMarket%2Bmapping.html) |
| 赛程变更 | [Fixture Change](https://docs.sportradar.com/uof/data-and-features/messages/event/fixture-change) |
| 体育赛事状态与生命周期 | [SportEventStatus](https://docs.sportradar.com/uof/sdk/languages/java/class-hierarchies/sporteventstatus) |
| 比赛状态工作流（足球） | [Soccer Match Status Workflow](https://developer.sportradar.com/soccer/docs/soccer-ig-match-status-workflow) |
| FAQ | [UOF FAQ](https://docs.sportradar.com/uof/support-and-history/faq/uof-faq) |
